## CSS Architecture Notes

### Theme Variable Migration

The current index.css defines all color variables inside `@theme`. For dark mode to work, these must be restructured:

1. Move ALL color variables from `@theme` to `@layer base` under `.dark` selector
2. Add `:root` selector in `@layer base` with light theme values
3. Keep only non-theme-varying tokens in `@theme` (font families, radius values)
4. Add `@custom-variant dark (&:where(.dark, .dark *));` after the Tailwind import

### Recharts Theming

All Recharts color values use CSS variable references (`var(--color-*)`) in SVG stroke/fill attributes. This ensures automatic theme adaptation without re-render. If any Recharts sub-element does not respect CSS variables in SVG context, fall back to resolving the variable value at render time via `getComputedStyle`.
