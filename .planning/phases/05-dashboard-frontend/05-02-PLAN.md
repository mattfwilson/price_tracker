    **StatusDot.tsx:**
    Props: { status: "ok" | "error" | "running" | "paused" }
    Renders a filled circle (w-2.5 h-2.5 rounded-full) with color mapping:
    - "ok" -> bg-emerald-500, label "OK"
    - "error" -> bg-red-500, label "Error"
    - "running" -> bg-amber-500 with optional animate-pulse, label "Running"
    - "paused" -> bg-zinc-400, label "Paused"
    Renders as: span with dot + text label in text-sm text-muted-foreground.

    Export a helper function deriveStatus(detail: WatchQueryDetailResponse | undefined, isScrapingLocal: boolean): "ok" | "error" | "running" | "paused":
    - If isScrapingLocal is true -> "running"
    - If detail and !detail.is_active -> "paused"
    - If detail and any retailer_url has latest_result === null (no scrape yet) -> "ok" (treat as fresh)
    - If detail is undefined (still loading) -> "ok"
    - Otherwise -> "ok" (all scrapes succeeded; there is no error field on LatestScrapeResult currently, so default to ok)

    **CardMenu.tsx:**
    Props: { queryId: number, queryName: string, isActive: boolean, onEdit: () => void, onScrapeNow: () => void, onDelete: () => void }
    Uses shadcn DropdownMenu component.
    Trigger: Button variant="ghost" size="icon" with MoreHorizontal icon from lucide-react (the three-dot icon). The trigger button MUST have onClick={(e) => e.stopPropagation()} to prevent card click propagation.
    Menu items:
    - "Edit" with Pencil icon -> calls onEdit
    - Pause/Resume (dynamic label: isActive ? "Pause" : "Resume") with Pause/Play icon -> calls usePauseQuery or useResumeQuery mutation with queryId
    - "Scrape Now" with RefreshCw icon -> calls onScrapeNow
    - DropdownMenuSeparator
    - "Delete" with Trash2 icon, className="text-destructive" -> calls onDelete

    **NewQueryCard.tsx:**
    Props: { onClick: () => void }
    Renders a Card with dashed border (border-dashed border-2), same height as other cards (min-h-[200px]).
    Content: centered Plus icon from lucide-react + "Add Query" text in text-muted-foreground.
    Hover: border becomes solid (hover:border-solid), cursor-pointer.

    **QueryCard.tsx:**
    Props: { query: WatchQueryResponse, onCardClick: (id: number) => void, onEdit: (id: number) => void, onDelete: (id: number) => void }
    This component:
    1. Calls useWatchQueryDetail(query.id) to get detail with latest_result data
    2. Manages local isScrapingLocal state (useState, defaults false)
    3. While detail is loading, shows Skeleton placeholders for price area

    Card layout (using shadcn Card):
    - Card root: cursor-pointer, onClick={() => onCardClick(query.id). If threshold breached: add "border-l-[3px] border-l-emerald-500" class.
    - Hover effect: "hover:shadow-md transition-shadow duration-150"
    - CardHeader area (no actual CardHeader component needed, just a div with padding):
      - Top row: query.name (font-heading text-xl font-bold, truncate) on left, URL count badge ("{N} urls" using Badge variant="secondary" with text-xs) and CardMenu on right
    - CardContent area:
      - Price section: If detail loaded and has prices:
        - Compute lowestPrice = Math.min(...detail.retailer_urls.map(u => u.latest_result?.price_cents).filter(Boolean))
        - Display formatPrice(lowestPrice) in font-heading text-3xl font-bold
        - Find the retailer_url whose latest_result.price_cents === lowestPrice, show its direction as deltaIcon + pct_change formatted as "+X.X%" or "-X.X%" in appropriate color (text-emerald-600 for lower, text-red-500 for higher, text-zinc-400 for unchanged/new)
        - If threshold breached (lowestPrice <= query.threshold_cents): show Badge with "Below threshold" in emerald variant (bg-emerald-500/10 text-emerald-600 border-emerald-500/20)
      - If detail loading: show Skeleton w-24 h-8 for price, Skeleton w-16 h-4 for delta
      - Threshold line: "Threshold: {formatPrice(query.threshold_cents)}" in text-sm text-muted-foreground
      - Status line: StatusDot component + formatRelativeTime(latestScrapedAt) where latestScrapedAt is the most recent scraped_at across all retailer_urls

    Scrape Now handler: set isScrapingLocal = true, call useScrapeNow mutation, on settle set isScrapingLocal = false and show toast("Scrape started for {query.name}").
