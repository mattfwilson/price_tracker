    **use-sse.ts:**
    Custom hook useAlertSSE() that:
    1. Gets queryClient from useQueryClient()
    2. Stores EventSource in useRef
    3. In useEffect:
       - Creates EventSource to "http://localhost:8000/alerts/stream"
       - Adds event listener for "alert" event type (NOT onmessage — backend sends named events)
       - On "alert" event: parse JSON data as AlertSSEPayload, then:
         a. Update unread count cache: queryClient.setQueryData(queryKeys.unreadCount, { unread_count: data.unread_count })
         b. Invalidate alerts list: queryClient.invalidateQueries({ queryKey: queryKeys.alerts })
         c. Show toast via sonner: toast(`${data.watch_query_name} — ${formatPrice(data.price_cents)} at ${data.retailer_name}`, { description: "Below threshold!", duration: 5000 })
       - On error: close EventSource, setTimeout to reconnect after 3000ms
       - Cleanup: close EventSource on unmount
    4. The queryClient from useQueryClient() is stable across renders, safe in effect closure
